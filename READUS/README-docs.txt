Документацию можно прочесть по 2 ярлыкам "Docs-1" и "Docs-2". В первом случае документация сделана при помощи Sphinx и выполнена в виде сайта, во втором случае сделана при помощи Pycco и в виде отдельных html страниц, которые расположены в соответствующей иерархии


Пример создания документации при помощи pycco:
	 pycco NeuralNetwork/**/**/**/**/**/*.py -p через Git Bash/shell в корневой папке проекта

	 В папке "Tools" в корне, есть скрипт .sh для автоматического создания doc.
